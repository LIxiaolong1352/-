### 重装系统后恢复方法
- OpenClaw 安装在 `C:\Users\Administrator\AppData\Roaming\npm\node_modules\openclaw`
- 工作目录在 `C:\Users\Administrator\.openclaw\workspace`
- 恢复步骤：
  1. 重装Node.js
  2. 运行 `npm install -g openclaw`
  3. 恢复 workspace 文件夹（从备份或GitHub克隆）
  4. 重新安装 skills: `npx clawhub install find-skills stock-analysis multi-search-engine weather playwright self-improvement skill-creator healthcheck`
- 重要：定期备份 `workspace` 文件夹到GitHub或其他地方

## 论文写作技能

### 已学习的范文
- 《河北省普通高校篮球运动员运动损伤流行病学调查及致因研究》（高万钧）
- 类型：体育学硕士论文
- 字数：约5万字
- 结构：摘要→引言→文献综述→研究方法→结果分析→致因分析→结论建议

### 本科论文标准格式
**前置部分**：封面、中英文摘要、关键词、目录
**正文部分**：引言、文献综述、研究对象与方法、结果与分析、结论与建议
**后置部分**：参考文献、附录、致谢

### 核心写作套路
1. **文献综述**：按主题分类，不是简单罗列
2. **研究方法**：文献法+问卷法+统计法（SPSS/Excel）
3. **数据分析**：描述统计+卡方/t检验+相关性分析
4. **图表呈现**：三线表、柱状图、饼图
5. **参考文献**：期刊[J]、书籍[M]、学位论文[D]，按出现顺序编号

### 写作效率技巧
- 建立模板框架，复制替换内容
- 同一主题换对象：A大学→B大学
- 数据生成要合理，参考真实论文范围
- 第一篇慢，后面流水线

### 注意事项
- 问卷要做信效度检验
- 引用用上标[1]，参考文献按顺序排列
- 查重：用自己的话转述，数据自己生成
